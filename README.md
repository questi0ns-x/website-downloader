---

# 🇪🇸 Español

## 📌 Descripción

**Netlifes Web Downloader** es una herramienta avanzada desarrollada en Python que permite descargar sitios web completos de forma automática.

El programa rastrea páginas HTML, descarga recursos estáticos y organiza todo el contenido en carpetas separadas por tipo de archivo.

Ideal para:

* Backups de páginas web
* Desarrollo offline
* Clonación de sitios estáticos
* Análisis web
* Archivado de contenido
* Investigación y testing

---

## ✨ Características

* 🌍 Descarga completa de sitios web
* 🖥️ Interfaz CLI moderna e interactiva
* 📂 Organización automática por carpetas
* 🖼️ Descarga de imágenes, CSS, JS, fuentes y vídeos
* ⚡ Modos rápido, normal y profundo
* 🔧 Configuración personalizada
* 📄 Guardado automático de HTML
* 🔁 Sistema de reintentos automático
* 📊 Estadísticas detalladas de descarga
* 🎨 Interfaz visual con colores
* 🛡️ Manejo avanzado de errores

## 🚀 Instalación

### 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/TU-USUARIO/netlifes-web-downloader.git
cd netlifes-web-downloader
```

### 2️⃣ Instalar dependencias

```bash
pip install requests beautifulsoup4 tqdm colorama
```

---

## ▶️ Uso

Ejecuta el programa con:

```bash
python web_downloader.py
```
